# Visualisation-of-algortihm-BCS-

## Initial Settup

To run this project, you must have java, and processing installed and added to your project.
From here there are two different routes, based on your operating system

Navigate to the project structure of your IDE. From here navigat to (local) libraries. In here you want to add the processing library. 
There is a specific path to get the wanted file:

### mac
Applications/processing/contents -> app -> resources -> core -> library
(You might need to double click the app icon for processing in your finder, to access the files)

### windows
libraries-> + icon -> java -> proccesing.app.ressources.core -> library

#### Finallly
Add the library to your libraries in your IDE.

We recommend using Intellej, which is what IDE this software has been programmed in.

## other libraries

### dashed lines 
  Download the following library https://github.com/garciadelcastillo/-dashed-lines-for-processing-/tree/master
  specifically go to dist, download the zip file and excstract.
  Now you can add the following file to your libraries, just how you added processing:
  dashedlines/libraries/dashedlines.jar
  

## how to run the program
Run Main.

A pop up will show up. This is the program for whihc the user can interact with.

- Pressing p will enable the debug feature.
- Pressing f11 will make the program full screen.
